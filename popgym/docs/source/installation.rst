
.. _installation:

Installation
------------


To use the POPGym environments, install it using pip:
From stable

.. code-block:: bash

    pip install popgym

Or from source

.. code-block:: bash

    git clone https://github.com/smorad/popgym
    cd popgym
    pip install .


To install the baselines (memory models) and RLlib

.. code-block:: bash

    pip install popgym[baselines]

Or from source

.. code-block:: bash

    git clone https://github.com/smorad/popgym
    cd popgym
    pip install ".[baselines]"
